# GIT Installation
---
## Windows

### Option 1: Official Installer (Recommended)

1. Go to [git](**git-scm.com**) 
2. Download **Git for Windows**
3. Run the installer
4. During setup:

   * Keep default options unless you know what you want to change
   * Choose **‚ÄúGit from the command line and also from 3rd-party software‚Äù**
5. Finish installation

Verify:

```bash
git --version
```

---

## macOS

### Option 1: Using Homebrew (Recommended)

If you have Homebrew:

```bash
brew install git
```

### Option 2: Official Installer

1. Download from [git](**git-scm.com**) 
2. Run the `.pkg` file

Verify:

```bash
git --version
```
